# Страхование - Android App

Это приложение является WebView оберткой для сайта [polissales.ru](https://polissales.ru).

## Сборка
Проект уже содержит всё необходимое, включая подпись `release.keystore`.
Чтобы собрать готовый APK (для прямой установки) или AAB (для публикации в RuStore):

1. Загрузите эту папку в свой репозиторий GitHub.
2. Сборка запустится автоматически через **GitHub Actions**.
3. Готовые файлы `.apk` и `.aab` можно будет скачать в разделе **Actions** -> **Artifacts** (на странице сборки внизу).

## Офлайн режим
При отсутствии подключения к сети отображается локальный экран с кнопкой "Обновить страницу". При появлении интернета страница автоматически загрузится по нажатию.
